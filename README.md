# 💳 Subscription Cost Tracker

A clean, responsive React web app to track all your subscriptions and instantly see your monthly and yearly spending.

**Built by:** Pasumarthy Abhinav — pasumarthyabhinav955@gmail.com

---

## ✨ Features

- **Add subscriptions** with name and monthly cost (₹)
- **Delete** any subscription instantly
- **Live totals** — monthly and yearly costs update in real time
- **Most Expensive** — the priciest subscription is automatically highlighted 👑
- **Smart icons** — auto-detects Netflix, Spotify, GitHub, Adobe, and more
- **localStorage persistence** — data survives page refreshes
- **Responsive** — works on mobile, tablet, and desktop

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Build for Production

```bash
npm run build
```

## ☁️ Deploy on Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Click **Deploy** — no config needed!

## 🛠 Tech Stack

- [React](https://react.dev) — UI with functional components and hooks
- [Vite](https://vitejs.dev) — Lightning-fast dev server & build tool
- Vanilla CSS — Custom design system, no frameworks
- localStorage — Zero-backend persistence
